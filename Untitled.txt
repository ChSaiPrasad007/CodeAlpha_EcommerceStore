bout this learning activity
What if AI could pull updated and accurate information in real time before providing responses? Retrieval-augmented generation (RAG) makes this possible. In this module, you’ll learn what RAG is and identify its key use cases. Next, you’ll explore the benefits of using RAG to enhance the knowledge base of an AI model. Finally, you’ll find out the key steps in the RAG workflow.

 

What you’ll learn

After completing this module, you should be able to:

Identify the key use cases for RAG
Explain the benefits of using RAG to expand the knowledge base of a large language model
Describe the key steps in a RAG workflow
 

Lessons

An overview of retrieval-augmented generation
Benefits of implementing RAG
How does RAG work?
Simulation: Enhancing a large language model using RAG


About this learning activity
Artificial intelligence (AI) agents are pushing the boundaries of automation and decision making to transform the way you work. In this module, you’ll learn about AI agents, how they perform tasks, and the different types of AI agents that exist. You’ll learn how to identify key use cases where AI agents improve efficiency and categorize them based on how they perform tasks. Finally, you’ll learn how AI agents work and where they’re making an impact today.

 

What you’ll learn

After completing this module, you should be able to:

Identify the key use cases for AI agents
Categorize AI agents based on their characteristics and applications
Describe how AI agents perform tasks
 

Lessons

Overview of AI agents
AI agent types and their applications
How do AI agents work?
Simulation: Personalizing fitness with an AI agent

The next evolution of AI focuses on AI agents collaborating, competing, and adapting together. In this module, you learn about multiagent systems and how they differ from single-agent systems. You explore how agents interact and work together and discover the ways these systems solve complex problems across different industries. Through practical examples, you uncover the power of agents collaborating to create innovative solutions.

What you’ll learn

After completing this module, you should be able to:

Distinguish between multiagent systems and single-agent systems
Categorize multiagent systems based on how individual agents interact with each other
Identify the benefits and use cases of multiagent systems across industries
 

Lessons

Introduction to multiagent systems
Types of multiagent systems
Applications of multiagent systems
Simulation: Automating procurement with multiagent systems
